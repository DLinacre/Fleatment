## 1. STRATEGIC OVERVIEW

**Fleatment** is an open-source, client-side web application and live UK cat flea & tick treatment index developed by **David Linacre**. It addresses a critical problem for cat owners: comparing real delivered prices (including UK VAT and standard shipping) and selecting safe, veterinary-appropriate parasite treatments.

Key features implemented in v1.0.0:
1. **Automated Live Data Auto-Fetching**:  fetches  on page load to automatically calculate true per-dose unit costs across top verified UK retailers (Pets at Home, Animed Direct, Pet Drug Online, VioVet, Amazon UK).
2. **Interactive Household Savings Calculator**: Multi-cat cost modeling predicting monthly and annual expenses.
3. **Veterinary Ingredient Matrix**: Comparative breakdown of Imidacloprid, Fipronil, Nitenpyram, and (S)-methoprene.
4. **Critical Permethrin Toxicity Warning**: Clear safety alert warning cat owners about fatal canine pyrethroid toxicity.
